# shadcn/ui React Router app

This is a single React Router app using shadcn/ui components.

## Adding components

To add components to your app, run the following command at the root of the frontend:

```bash
npx shadcn@latest add button
```

This will place the ui components in the `components` directory.

## Using components

To use the components in your app, import them from the local alias.

```tsx
import { Button } from "@components/button";
```
